# Storyboard

## Visual Planning for Video Production

This document provides visual planning and storyboard frames for the video production.

### Frame 1: Opening - The Heavy Mic (00:00:00 - 00:00:10)
- **Shot Type:** Wide shot
- **Visual Elements:** Host holding massive golden microphone
- **Mood:** Dramatic, symbolic
- **Camera Movement:** Static establishing shot

### Frame 2: The Pivot (00:00:10 - 00:00:35)
- **Shot Type:** Medium close-up
- **Visual Elements:** Host speaking directly to camera, people struggling in background
- **Mood:** Empathetic, determined
- **Camera Movement:** Slight push-in

### Frame 3: Statues vs. Mercury (00:00:35 - 00:00:48)
- **Shot Type:** Split screen / Comparison shot
- **Visual Elements:** 
  - Left: Frozen stone statues (static rules)
  - Right: Liquid mercury character (dynamic AI)
- **Mood:** Contrasting, transformative
- **Camera Movement:** Pan between two scenes

### Frame 4: The Clone Lab (00:00:48 - 00:03:52)
- **Shot Type:** Screen capture overlay
- **Visual Elements:** Terminal window showing `git clone` command
- **Mood:** Technical, educational
- **Camera Movement:** Zoom in on terminal

### Frame 5: The Evolution (00:03:52 - 00:05:50)
- **Shot Type:** Wide shot with animation overlay
- **Visual Elements:** 3D printer creating a copy
- **Mood:** Futuristic, innovative
- **Camera Movement:** Slow orbit around printer

### Frame 6: The Nursery (00:05:50 - 00:06:36)

- **Mood:** Family-oriented, caring
- **Camera Movement:** Gentle push-in
- **Lighting:** Soft, warm tones

### Frame 7: The Crystal Ball (00:06:36 - 00:08:26)
